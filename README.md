📌 Project Description
This is a C# console chatbot designed to educate users about cybersecurity topics such as phishing, password safety, and safe browsing.

⚙️ Features
- Voice greeting (WAV file)
- ASCII art logo
- Personalised user interaction
- Input validation
- Cybersecurity responses
- Coloured console interface

▶️ How to Run
1. Open project in Visual Studio
2. Build the solution
3. Run the program
